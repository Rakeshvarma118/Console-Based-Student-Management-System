# Console-Based-Student-Management-System
### Java | JDBC | MySQL | Console Application

A **menu-driven Employee Management System** developed using **Java and JDBC**, with **MySQL** as the backend database.  
This project focuses on **secure database operations**, **clean code structure**, and **strong backend fundamentals**, making it ideal for **Java fresher interviews**.

---

##  About the Project

The Employee Management System is a console-based application designed to manage employee data efficiently.  
It allows users to perform **CRUD operations** (Create, Read, Update, Delete) through a simple and interactive menu-driven interface.

The project emphasizes:
- JDBC architecture and workflow
- Secure SQL execution using parameterized queries
- Modular and maintainable code design

---

##  Features

- Insert new employee records  
- Update existing employee details  
- Fetch a single employee record by ID  
- Fetch all employee records  
- Delete employee records  
- Menu-driven console interface  
- Protection against SQL Injection using `PreparedStatement`

---

##  Tech Stack

- **Java**
- **JDBC (Java Database Connectivity)**
- **MySQL**
- **Scanner (Console Input)**
